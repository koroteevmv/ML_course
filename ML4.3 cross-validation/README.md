##### Stratified k-Fold

–ú–µ—Ç–æ–¥ stratified k-Fold ‚Äî —ç—Ç–æ –º–µ—Ç–æ–¥ k-Fold, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–π —Å—Ç—Ä–∞—Ç–∏—Ñ–∏–∫–∞—Ü–∏—é –ø—Ä–∏ —Ä–∞–∑–±–∏–µ–Ω–∏–∏ –Ω–∞ —Ñ–æ–ª–¥—ã: –∫–∞–∂–¥—ã–π —Ñ–æ–ª–¥ —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—Ä–∏–º–µ—Ä–Ω–æ —Ç–∞–∫–æ–µ –∂–µ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ –∫–ª–∞—Å—Å–æ–≤, –∫–∞–∫ –∏ –≤—Å—ë –∏—Å—Ö–æ–¥–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ.
–¢–∞–∫–æ–π –ø–æ–¥—Ö–æ–¥ –º–æ–∂–µ—Ç –ø–æ—Ç—Ä–µ–±–æ–≤–∞—Ç—å—Å—è –≤ —Å–ª—É—á–∞–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä, –æ—á–µ–Ω—å –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏—è –∫–ª–∞—Å—Å–æ–≤.

–°–æ–∑–¥–∞—ë–º —Å—Ç—Ä–∞—Ç–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ k-–±–ª–æ—á–Ω–æ–µ —Ä–∞–∑–±–∏–µ–Ω–∏–µ (StratifiedKFold):
